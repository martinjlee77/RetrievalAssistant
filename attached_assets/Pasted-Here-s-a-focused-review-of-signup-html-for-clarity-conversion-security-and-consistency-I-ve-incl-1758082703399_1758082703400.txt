C) Business email blocklist (client-side) Add before form submit handler: const freeDomains = [ 'gmail.com','yahoo.com','outlook.com','hotmail.com','live.com','aol.com','icloud.com', 'mail.com','proton.me','protonmail.com','yandex.com','gmx.com','zoho.com','me.com' ]; function isBusinessEmail(email) { const domain = email.split('@')[1]?.toLowerCase().trim(); return domain && !freeDomains.includes(domain); } In submit handler, after regex validation: if (!isBusinessEmail(formData.email)) { showMessage('Please use a valid business email address (no gmail, outlook, yahoo, etc.)', 'error'); return; } Note: Enforce this server-side too.
